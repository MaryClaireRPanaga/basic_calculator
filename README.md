# Basic Calculator Project

A Python application that implements a functional calculator using Object-Oriented Programming (OOP) principles. This project focuses on structured code, clean logic, and reliable arithmetic operations while ensuring a user-friendly console interface.

## 🚀 Features
- Basic Arithmetic: Handles standard operations including addition, subtraction, multiplication, and division.
- Input Validation: Ensures user inputs are handled correctly during calculator operations.
- Interactive Console Interface: Provides a continuous loop allowing users to perform multiple calculations without restarting the program.
- Clean OOP Architecture: Built using modular class structures to separate calculator logic from the main execution flow.

## 🛠️ Built With
- Language: Python 3.x
- Version Control: Git and GitHub
- Development Environment: VS Code

## 📁 Code Structure
The project is organized to keep the calculator logic clean and maintainable:
- main.py — The primary entry point that runs the program, manages the execution loop, and handles user input.
